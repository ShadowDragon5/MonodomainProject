# Final Project: Monodomain Equation

\begin{align*}
	\frac{\partial u}{\partial t} - \nabla \cdot \Sigma \nabla u + f(u) & = 0   & \text{in } \Omega \times I,          \\

	u                                                                   & = u_0 & \text{in } \Omega \times \{0\},
\end{align*}
